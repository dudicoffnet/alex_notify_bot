1. Установите зависимости: pip install -r requirements.txt
2. Вставьте свои BOT_TOKEN и ADMIN_ID в .env или Railway переменные.
3. Залейте проект в GitHub → Railway.
4. Бот:
   - 10:00 (Минск) — PDF в Telegram
   - 23:00 (Минск) — PDF в Telegram
   - /force — PDF и ZIP сразу
   - heartbeat — каждые 5 мин
