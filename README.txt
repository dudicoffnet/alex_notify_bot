
Бот-уведомитель (aiogram 3) с автоотчётами и полезными командами.

Команды:
/ping — проверка связи
/report — текстовый отчёт сразу
/pdfreport — PDF-отчёт (файл)
/testnow — отправить отчёт немедленно
/settime HH:MM HH:MM — обновить расписание автоотчётов без деплоя
/backup — выгрузка ZIP с настройками (без .env)
/health — самопроверка: токен, ID, расписание, ближайшие задания

Расписание по умолчанию: 10:00 и 23:00 (Europe/Minsk). В runtime хранится в schedule.json.

ENV:
BOT_TOKEN=...
ADMIN_ID=... (число)
